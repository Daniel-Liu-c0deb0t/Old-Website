About the Website

This website was created as a cool, artistic side project. It can also act like a cool graphical portfolio for my web design skills. As such, only the front page exists and most links and images goes to my other, actual site on Wordpress. Visit it at c0deb0t.wordpress.com.


This site was created using HTML 5, CSS, JavaScript (obviously). JQuery was used to make JavaScripting easier. Font Awesome was also used for some icons. Images found on Pexels were used as inspiration. Google's Fonts API provided an easy way to incorporate cool fonts into the website. Google's CDN services were wonderful because I don't need to download anything. Everything is served right to the user with just an HTML link.


Two websites that provided me with lots of inspiration and got me to start creating this website was darli-fra.jp and fate-extra-lastencore.com. Also, I chose to focus on roses and the color red for no reason in the design. It just looks elegant.
